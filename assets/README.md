# Assets Directory

This directory contains assets for the Long-Running Agent skill:

## Files
- `icon.svg` - Skill icon for Agent Skills platforms (when available)
- `banner.png` - Repository banner for GitHub (when available)
- `demo.gif` - Demo animation showing the skill in action (when available)

## Usage
Assets in this directory are used by:
- Agent Skills platforms for skill discovery and presentation
- GitHub repository for better visual presentation
- Documentation and tutorials

## Creating Assets
If you'd like to contribute assets:
1. Icon should be SVG format, 64x64px minimum
2. Banner should be PNG format, 1200x630px for GitHub social preview
3. Demo should be GIF format, under 10MB

All assets should follow the MIT license and be suitable for open source distribution.
